Kegiatan Praktikum Tiap Minggu

Berikut merupakan detail kegiatan yang dilakukan pada Praktikum SIG Web Tiap Minggu

Minggu-1, 2, dan 3

BAB : Latihan Membuat Halaman HTML Sederhana dan CSS

Pada latihan-1 kita membuat tulisan miring, tulisan tebal, memberi warna pada tulisan, menyantumkan link dan menghubungkannya agar dapat dibuka, memunculkan gambar kucing, membuat tabel sederhana, dan membuat form yang dapat diisi dengan menginputkan sesuai instruksi. Selanjutnya pada latihan-2 membuat halaman sederhana mengunakan HTML dan CSS untuk membuat halaman "City Gallery". Yang dilakukan adalah membuat heading berwarna, memasukkan gambar yang dibutuhkan, dan menambahkan deskripsi dari gambar tersebut. Yang ketiga membuat Resume atau Curriculum Vitae menggunakan template yang sudah disediakan oleh website WIX.

Minggu-4

BAB :

Pada praktikum Minggu ke-4 ini, kita melakukan pembuatan halaman HTML sederhana tentang Hitungan Fungsi faktorial, Menampilkan Bilangan Prima dari angka masukan, Menampilkan Bilangan Fibonacci dari angka masukan, validasi nama umur, dan validasi password.

Minggu-5

BAB : BootStrap dan GitHUb

Pada Praktikum Minggu ke-5 ini, kita melakukan pembuatan Resume atau Curriculum Vitae menggunakan template dari BootStrap. Melakukan pengunduhan template yang diinginkan, kemudian membuka zip dan mengisikan hal-hal yang ingin dimasukkan pada file html nya. Kita membuat landing page sebagai halaman pembuka dari website yang kita buat, kemudian emngisikan nama, biodata yang diperlukan, pengalaman, list tugas dalam bentuk tabel, dan contact.

Minggu-6

BAB : Membuat Webmap Leaflat

Pada praktikum Minggu ke-6, kita melakukan pembuatan WebMap dengan menggunakan LeafletJS dilengkapi dengan pulgin pendukung. Hasil dari peta yang kami buat adalah menampilkan rute dari kos atau tempat tinggal masing-masing menuju Teknik Geodesi dengan menggunakan Control leaflet-routing-machine selain itu, juga menambahkan point dari lokasi tiap departemen yang ada di fakultas teknik. Sebagai pelengkap informasi, kami juga menambahkan popup yang berisi gambar dan teks tiap titik lokasi

Minggu-7

BAB : Membuat Webmap Leaflat Lanjutan dengan GEOJSON

Pada praktikum Minggu ke-7, kita melakukan pembuatan WebMap LeafletJS lanjutan dengan menambahkan file GEOJSON. Membuka QGIS kemudian melakukan ekspor dari filw shp wilayah yang dibutuhkan kemudian ekspor dalam format GEOJSOM dan merubah menjadi wgs84. Memasukkan hasil koordinat GEOJSON sebelumnya dnegan menambahkan variable pada bagian atas
